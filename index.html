Start Sublime Text!

Install Package Control

Sublime Text plugins (called Packages) can be installed in multiple ways. Easiest way is Package Control.

  • Go to http://packagecontrol.io/
  • Copy the code for your version of ST
  • In ST, press Ctrl + `, paste code, press Enter
  • ST asking for resart multiple times is normal!

Install Xdebug Client

Now that we have Package Control, installing any plugin is just a few keystrokes away!

  • Go to Preferences → Package Control or press Ctrl + Shift + P
  • Type Install, select Install Package, hit Enter
  • Type xdebug, hit Enter
