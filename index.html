Install Package Control

Sublime Text plugins (called Packages) can be installed in multiple ways. Easiest way is Package Control.

  • Go to http://packagecontrol.io/
  • Copy the code for your version of ST
  • In ST, press Ctrl + `, paste code, press Enter
  • ST asking for resart multiple times is normal!

Install Xdebug Client

Now that we have Package Control, installing any plugin is just a few keystrokes away!

  • Go to Preferences → Package Control or press Ctrl + Shift + P
  • Type Install, select Install Package, hit Enter
  • Type xdebug, hit Enter

Xdebug

Let's Install Xdebug

  • Ubuntu/Mint: sudo apt-get install php5-xdebug
  • Others: PECL.
  • Open php.ini or 20-xdebug.ini

Paste these lines at end of config file 

            
                xdebug.remote_enable = on
                xdebug.remote_connect_back = on
                xdebug.idekey = "sublime.xdebug"

Save config file, restart apache

Debugging Terms

  • Breakpoint: Code will pause here. You place it in text editor/IDE.
  • Step Over: Current line will be evaluated, debugger will stop at next line.
  • Step Into: Current line will be evaluated, debugger will enter any function called on that line.
  • Step Out: Debugger will complete any function you have stepped into.
